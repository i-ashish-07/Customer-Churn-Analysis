# 📊 Customer Churn Analysis – Power BI & SQL Project

## 📌 Project Overview

This project focuses on analyzing customer churn to identify **key drivers, high-risk segments, and actionable business insights**.

Using SQL for data preparation and Power BI for visualization, the project provides a complete view of:

* Customer behavior
* Service usage patterns
* Churn drivers
* Revenue impact

The goal is to help businesses **reduce churn and improve customer retention strategies**.

---

## 📂 Dataset Information

* Total Records: **6,418 customers** 
* Features: **30+ columns** including demographics, services, contract, and revenue
* Key Fields:

  * Customer demographics (age, gender, state)
  * Services (internet, streaming, support)
  * Financials (monthly charges, total revenue)
  * Customer status & churn reason

---

## 🛠️ Tech Stack

* **Power BI** → Dashboard & visualization
* **SQL (PostgreSQL)** → Data cleaning & analysis
* **DAX** → KPI calculations & dynamic insights
* **Excel/CSV** → Data source
